Página de Boas-vindas

Essa Página é apenas para dar as boas-vindas ao site.

Isso é um subtítulo

$$ax^2 + bx + c = 0$$

Encontrar as raízes da função, temos: $$\Delta = b^2 - 4ac$$

Logo, $$x^{1} = \frac{-b + \sqrt{\Delta}}{2a}$$ $$x^{2} = \frac{-b - \sqrt{\Delta}}{2a}$$

Portanto,

Para $\Delta < 0$: Sem raízes reais

Para $\Delta = 0$: $x^{1} = x^{2}$

Para $\Delta > 0$: $x^{1} \neq x^{2}$
